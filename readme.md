# Dan Frameworks

## Dependencies

```
npm install -g 
```


```
npm install -g gulp
```

## Usage

First make a new directory, and `cd` into it:
```
mkdir my-new-project && cd $_
```

import modules:
```
npm install 
```



To see the result launch:
```
gulp
```

## Gulp tasks

* ```gulp```
* ```gulp build```
* ```gulp build:server```
* ```gulp deploy```
